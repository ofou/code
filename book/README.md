# Open Source Machine Learning Book

> *“Sometimes it is the people no one can imagine anything of who do the things no one can imagine.”*
> ― Alan Turing

Inspired by books like *Code* [-@petzold2000code], *Nand2Tetris* [-@nisan2021elements], the *[42 School]*, *[Karpathy]*'s projects, *[Tinygrad]*'s philosophy, and a [hacker's curriculum], this book aims to build understanding from first principles in Computer Science from the Assembly code to {abbr}`XAI (Explainable Artificial Intelligence)` by building real projects, be up-to-date, free and open source, and using a top-down approach.
This book is also intended to be a physical or online companion to the forthcoming [YouTube] series.
